# Hotel Management System (SWP391)

## Description
This project is a Hotel Management System designed to handle various hotel operations and management tasks.

## Technologies Used
- **Backend:**
  - Java
  - Java Servlets
  - MySQL (Database)
- **Frontend:**
  - HTML
  - CSS
  - JavaScript
  - Bootstrap
  - SweetAlert.js
  - Chart.js
